WatCard Balance

WatCard Balance is a Chrome extension built with primarily JavaScript. It uses HTTP requests to retrieve spending logs for the user's WatCard so they can better manage their spending. The extension provides balance information and graphical representations of their spending habits.

Languages: JavaScript, JQuery

Libraries: HighCharts

Created for EngHack 2015.

UWFlow Plus

UWFlow Plus is a Chrome extension is an extension which pulls information from the UWFlow API about courses and formats it in a hover window for easy access.

APIs: UWFlow

Sorting Algorithm Visualizer

I made this project shortly after becoming interested in algorithms and efficiency. It uses multithreading to give a live, real-time comparison of sorting algorithms for a given set of numbers.

Languages: Java

APIs: Swing

Metashift

A project which looks for shifts in the "metagame" of League of Legends from one patch to another. We used a Django/SQLite backend to process hundreds of thousands of matches from the Riot Games API. The front-end logic aggregates data based on input criteria and presents the data using HighCharts.

Languages: JavaScript

Libraries: Highcharts

APIs: Riot Games

Created for the second Riot Games API Challenge.
