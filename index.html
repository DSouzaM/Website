I wrote this application to scrape co-op employment information and gain some insight into the overall makeup of co-op employment at UWaterloo. It uses Python to collect information and manage the database, and has Flask as a backend. The frontend uses HighCharts.

WatCard Balance

WatCard Balance is a Chrome extension which helps students manage their spending. It retrieves spending logs for the user's WatCard, providing balance information and graphical representations of their spending habits. Built for EngHack 2015 using jQuery and HighCharts.

GitHub Chrome Web Store

UWFlow Plus

UWFlow Plus is a Chrome extension which makes it easy to get information about your courses. It pulls information from UWFlow into your course calendar, giving you information about courses at a glance.

Metashift

Metashift is a project which looks for shifts in the "metagame" of League of Legends from one patch to another. We used a Django backend to process hundreds of thousands of matches from the Riot Games API, and then used HighCharts to display the results. Created for the second Riot Games API Challenge.

GitHub Live Demo
